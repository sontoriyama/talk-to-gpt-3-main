## Running Locally

1. Clone the repository

```bash
git clone https://github.com/davila7/talk-to-gpt-3
cd talk-to-gpt-3
```
2. Install dependencies

These dependencies are required to install with the requirements.txt file:

* pytest 
* openai 
* pyttsx3 
* SpeechRecognition 
* pyaudio
* openai whisper 

```bash
pip install -r requirements.txt
```
1. Run

```bash
python app.py
```
